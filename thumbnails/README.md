# thumbnails/ — daily thumbnail drop folder

Every morning at 6 AM the bot writes `TODAY_TOPIC.txt` here (and to your Google
Drive folder if configured) telling you the day's topic and the exact filename to use.

## To use your own custom thumbnail (optional)
1. Read `TODAY_TOPIC.txt` — it gives you the **slug**, e.g. `bermuda_triangle`.
2. Make a **1280×720 PNG** thumbnail.
3. Drop it here (or in the shared Drive folder) named exactly: `bermuda_triangle.png`.
4. At 4:30 PM the bot finds it by slug and uses it.

## If you drop nothing
The bot auto-generates a dramatic thumbnail with free AI tools and a bold Hinglish
hook overlay — so upload is **never** blocked. Custom is optional; automatic is the default.
