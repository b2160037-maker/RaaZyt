# assets/sfx/ — sound effects (optional, free)

Drop short `.wav`/`.mp3` effects here for extra punch:
- `whoosh.wav` — transition whoosh on hard cuts
- `heartbeat.wav` — tense-beat pulse

**Free sources:** Pixabay (https://pixabay.com/sound-effects/) and Freesound (https://freesound.org).

These are optional — the signature sting already carries the core sound identity,
so the pipeline runs fine with this folder empty.
