# assets/music/ — background music (free, royalty-free ONLY)

Drop one or more cinematic/suspense tracks here as `.mp3` or `.wav`.
The pipeline picks one at random per video and ducks it well under the voice.

**Where to get free, royalty-free tracks:**
- Pixabay Music — https://pixabay.com/music/ (search "suspense", "cinematic", "dark")
- YouTube Audio Library — https://studio.youtube.com → Audio Library
- Free Music Archive — https://freemusicarchive.org
- Incompetech (Kevin MacLeod) — https://incompetech.com/music/ (credit as required)

**Important:** only use tracks licensed for commercial/YouTube use to avoid copyright strikes.
If this folder is empty the video still renders — just with no background music.
